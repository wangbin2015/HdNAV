HDMap.TileLayer（Example）
//创建地图对象
var map = HDMap.map('map', {
    center: [38.92705, 121.6580833],
    zoom: 10
});
//增加栅格图层
HDMap.tileLayer('http://175.25.246.208:8082/maptile-ais/maptile?z={z}&x={x}&y={y}').addTo(map);

创建（Creation）

map创建 说明
HDMap.tileLayer( <String> urlTemplate, <TileLayer options> options? ) 实例化一个砖层对象给定URL模板和选择一个选择对象。

选项（Options）

选项 类型 说明
minZoom Number 最小级别数
maxZoom Number 最大级别数
tileSize Number 切片尺寸（宽度和高度的像素值，假设切片是正方形的）
subdomains String or String[] 服务的子域。可以传递一个字符串（其中每一个字母都是一个子域名称）或是一个字符串数组。
errorTileUrl String 图片的URL给出加载错误的位置。
opacity Number 切片图层的透明度。
zIndex Number 切片图层明确的叠置顺序，默认此项不会被设置。
unloadInvisibleTiles Boolean 如果此项为true，在平移后所有看不到的切片都会被移除（用以更好地显示），在移动设备的webkit中默认是true，其他的默认为false。
updateWhenIdle Boolean 如果此项为false，在平移过程中新的切片将会载入，其他的在其后载入（用以更好地显示），在移动设备webKit中默认是true，其他默认false。
detectRetina Boolean 如果此项为true，并且用户是视网膜显示模式，会请求规定大小一般的四个切片和一个地区内一个更大的缩放级别来利用高分辨率。

事件（Events）

选项 类型 说明
loading Event 当切片图层开始加载切片时触发。
load Event 当切片图层加载完可见切片后触发。
tileload TileEvent 在加载切片时触发。
tileunload TileEvent 在切片被移除时触发（比如打开了unloadInvisibleTiles）。

方法（Methods）

选项 类型 说明
addTo() this 将图层加到地图上。
bringToFront() this 将此切片图层放到所有切片图层之上。
bringToBack() this 将此切片图层放到所有切片图层之下。
setOpacity() this 改变切片图层的透明度。
setZIndex() this 设置切片图层的叠放顺序。
redraw() this 清除所有的切片并重新向服务端申请他们。
setUrl() this 更新图层的URL模板并重绘他们。
