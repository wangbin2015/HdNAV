HDMap.Polygon（Example）
//创建地图对象
var map = HDMap.map('map', {
    center: [38.92705, 121.6580833],
    zoom: 10
});
//创建多边形
var polygon = HDMap.polygon([
    [38.92705, 121.7580833],
    [38.62705, 121.8580833],
    [38.93705, 121.8580833]]);
//捆绑popup
polygon.bindPopup("I am a polygon.").addTo(map);
//自适应地图窗口
map.fitBounds(polygon.getBounds());

创建（Creation）

多边形创建 说明
HDMap.polygon( <LatLng[]< latlngs, <Polyline options< options? ) 实例化一个多边形对象数组的地理点,选择一个选择对象(多线)一样。您还可以创建一个多边形孔通过一个数组的数组latlngs,第一latlngs数组代表外环其余代表了洞内。
