HDMap.Circle（Example）
//创建地图对象
var map = HDMap.map('map', {
    center: [38.92705, 121.6580833],
    zoom: 10
});
//创建圆对象
var circle = HDMap.circle(
    [38.92705, 121.6580833],
    5000, {color: '#f03', opacity: 0.7})
//捆绑popup
.bindPopup("I am a circle.")
//添加到map图层上

创建（Creation）

圆创建 说明
HDMap.circle( <LatLng> latlng, <Number> radius, <Path options> options? ) 实例化一个圆对象给定一个地理点,米半径,可选一个选项对象。

方法（Methods）

选项 类型 说明
getLatLng() LatLng 返回当前地理位置的圆。
getRadius() Number 返回当前半径的一个圆。单位是米。
setLatLng() this 设置一个圆圈的位置移动到新的位置。
setRadius() this 设置圆的半径。单位是米。
