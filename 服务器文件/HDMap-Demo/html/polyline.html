HDMap.Polyline（Example）
//创建地图对象
var map = HDMap.map('map', {
    center: [38.92705, 121.6580833],
    zoom: 10
});
//创建折线
var polyline = HDMap.polyline([
    [38.92705, 121.7580833],
    [38.62705, 121.8580833],
    [38.82705, 122.3580833]],
    {color: 'red'}).addTo(map);
//自适应地图窗口
map.fitBounds(polyline.getBounds());

创建（Creation）

折线创建 说明
HDMap.polyline( <LatLng[]< latlngs, <Polyline options< options? ) 实例化一个多段线对象数组的地理点,选择一个选择对象。

选项（Options）

选项 类型 说明
smoothFactor Number 决定每一个缩放级别上线段简化程度。如果大的话意味着更好的表现和看起来更光滑，小的话意味更准确地表示。
noClip Boolean 不允许线段裁剪。

方法（Methods）

选项 类型 说明
addLatLng() this 向线段添加一个点。
setLatLngs() this 用所给的地理点的数组替代线段上的点。
getLatLngs() LatLng[] 返回路径上的点组成的数组。
spliceLatLngs() LatLng[] 允许添加、移除和更改线段上的点。同Array#splice的语法一致。返回移除点组成的数组。
getBounds() LatLngBounds 返回线段的经纬度边界。
