HDMap.Popup（Example）
//创建地图对象
var map = HDMap.map('map', {
    center: [38.92705, 121.6580833],
    zoom: 10
});
//添加marker
var marker = new HDMap.Marker(latlng);
map.addLayer(marker);
//绑定popup
marker.bindPopup("<p>这里是大连</p>")
//打开popup
marker.openPopup();

创建（Creation）

map创建 说明
HDMap.popup( <Popup options> options?, <ILayer> source? ) 实例化一个弹出对象给出一个可选的选择对象,描述它的外观和位置和一个可选的源对象,用于标记弹出与ILayer参考引用。

函数构造器（Constructor）

选项 类型 说明
maxWidth Number （最大宽度）：弹出框的最大宽度。
minWidth Number （最小宽度）：弹出框的最小宽度。
maxHeight Number （最大高度）：设置后，如果内容超过弹出窗口的给定高度则产生一个可以滚动的容器。
autoPan Boolean （自动平移）：如果你不想地图自动平移来适应打开的弹出框，就设置其为false。
closeButton Boolean （关闭按钮）：控制弹出窗口中出现的关闭按钮。
offset Point （补偿值）：弹出窗口位置的补偿值。在同一图层中打开弹出窗口时对于控制锚点比较有用。
autoPanPadding Point （自动平移填补）：在地图视图自动平移产生后弹出窗口和地图视图之间的边缘。

方法（Methods）

选项 类型 说明
addTo() this 将弹出窗口添加到地图上。
openOn() this 将弹出窗口添加到地图上并将之前的一个关闭。与map.oenPopup(popup)方法相同。
setLatLng() this 设置弹出窗口打开的地理上的点位。
setContent() this 设置弹出窗口的HTML内容。
